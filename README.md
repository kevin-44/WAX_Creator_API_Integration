# Overview

The [WAX Creator API](https://github.com/worldwide-asset-exchange/wax-creator) allows developers to create NFTs (WAX Collectible Cards, WAX Stickers & WAX Digital Art) on the WAX Blockchain for free via external sources. NFTs must be approved by the support team, before minted on the WAX Blockchain, and sent to [your WAX ExpressTrade Inventory](https://trade.wax.io/inventory).
